Termos de Uso

Última atualização: 16/06/2026

1. Natureza do serviço

O Monitor Comunitário organiza informações públicas sobre desligamentos programados e gera alertas comunitários conforme dados cadastrados pelo usuário.

2. Não é canal oficial

O serviço não é operado pela Celesc e não substitui canais oficiais. As informações podem estar incompletas, indisponíveis, atrasadas, canceladas ou alteradas.

3. Responsabilidade do usuário

O usuário deve informar dados verdadeiros, não cadastrar dados de terceiros sem autorização e compreender que os alertas são indicativos, não garantias.

4. Publicidade

A interface pode conter espaços de publicidade local. Quando anúncios reais estiverem ativados, o carregamento de scripts publicitários dependerá das flags do sistema e da escolha de consentimento do usuário.

Nota para produção

Este texto é uma base técnica e informativa. Antes de uso público real, os documentos legais devem ser revisados por profissional jurídico.
