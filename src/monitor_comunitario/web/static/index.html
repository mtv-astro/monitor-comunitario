Como funciona

Da página pública ao alerta comunitário

  1. Coleta O sistema consulta os avisos públicos de desligamento programado.
  2. Persistência Cada aviso é salvo com hash para evitar duplicidade.
  3. Cruzamento Município, bairro e rua são comparados com os cadastros.
  4. Notificação Alertas aparecem no painel e podem evoluir para WhatsApp depois.
