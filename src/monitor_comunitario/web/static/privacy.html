Política de Privacidade

Última atualização: 16/06/2026

Esta Política de Privacidade explica como o Monitor Comunitário coleta, usa e protege dados pessoais informados por moradores que desejam acompanhar possíveis desligamentos programados de energia em sua região.

1. Quem somos

O Monitor Comunitário é um projeto independente de monitoramento de avisos públicos. Ele não representa a Celesc, não substitui canais oficiais e não garante a ocorrência, cancelamento ou alteração de desligamentos programados.

2. Quais dados coletamos

Podemos coletar nome, telefone/WhatsApp, município, bairro, rua, número, CEP e preferências de alerta.

3. Para que usamos os dados

Usamos os dados para manter o cadastro, comparar endereço com avisos públicos de desligamento programado, gerar notificações e auditar o funcionamento básico do sistema.

4. Publicidade e analytics por flags

O sistema possui flags para operar com anúncios e analytics desativados ou ativados. Quando ativados, scripts de terceiros só devem ser carregados conforme a escolha registrada no banner de consentimento.

5. Compartilhamento

Os dados cadastrais não são vendidos. Caso provedores de anúncios, analytics ou WhatsApp sejam ativados, esta política deve informar os terceiros envolvidos e suas finalidades.

6. Direitos do titular

A pessoa cadastrada pode solicitar acesso, correção ou exclusão dos dados. Em uma implantação real, o responsável pelo sistema deve disponibilizar canal de contato para essas solicitações.

Nota para produção

Este texto é uma base técnica e informativa. Antes de uso público real, os documentos legais devem ser revisados por profissional jurídico.
